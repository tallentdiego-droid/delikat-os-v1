# M6-008 --- Product Documentation Governance

## Recipe Documentation

-   Recipes are maintained as official operational documents.
-   Ingredient quantities are expressed using the units defined in each
    recipe.
-   Recipe yields are documented for production planning.

## Production Sequence

-   Complete PREP recipes before dependent kitchen recipes.
-   Use only standardized prepared components when specified.
-   Follow the documented preparation order.

## Portion Control

-   Portion products using the documented quantity.
-   Package products after portioning when required.
-   Label packaged products before storage.

## Warehouse Delivery

-   Finished PREP products are delivered to warehouse after packaging
    and labeling.
-   Warehouse receives products according to the documented storage
    process.

## Menu Consistency

-   Every finished menu item shall be prepared using its corresponding
    standardized recipe.
-   Final presentation is verified before service.
