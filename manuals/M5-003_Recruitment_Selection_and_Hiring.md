# M5-003 --- Recruitment, Selection and Hiring

## Recruitment

-   Recruitment begins after an approved personnel requisition.
-   Vacancies are published using the approved recruitment notice.
-   Applicants complete the standard employment application.
-   Applications are reviewed before interviews.

## Personnel Requisition

The requisition records: - Position requested - Department - Reason for
hiring - Date requested - Requesting manager - Required start date

## Selection Process

-   Review applications.
-   Schedule interviews.
-   Evaluate candidates using the approved interview form.
-   Verify required documentation.
-   Select the candidate that best meets the position profile.

## Interview

The interview form documents: - Personal information - Work experience -
Education - Availability - References - Interview observations - Hiring
recommendation

## Personnel File

A personnel file shall be maintained for every employee and include: -
Employment application - Supporting documentation - Evaluation records -
Disciplinary records - Vacation records - Leave records -
Confidentiality agreement - Other approved HR forms

## Hiring Rule

Employment documentation shall be completed before incorporation into
operations.
